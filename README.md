# Pizza_Sales_Dashboard
## Interactive Pizza Sales Dashboard built using SQL Server and Microsoft Excel. Features KPI analysis, sales trends, pizza category performance, and business insights through an interactive dashboard.
## Data Analytics Project | SQL Server | Microsoft Excel | Interactive Dashboard | KPI Reporting | Sales Analysis | Data Visualization | Business Intelligence

## Dataset used 
 - <a href="https://github.com/amanbaliyan112/Pizza_Sales_Dashboard/blob/main/pizza_sales_excel_file.xlsx">Dataset</a>

File:
- pizza_sales.csv

The dataset contains one year of pizza sales transactions, including order date, pizza category, pizza size, quantity sold, unit price, and total price.
