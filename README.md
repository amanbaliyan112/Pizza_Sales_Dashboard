# Pizza_Sales_Dashboard

## Data Analytics Project | SQL Server | Microsoft Excel | Power BI | SQL | Interactive Dashboard | KPI Reporting | Sales Analysis | Data Visualization | Business Intelligence

## Dataset used 
 - <a href="https://github.com/amanbaliyan112/Pizza_Sales_Dashboard/blob/main/pizza_sales_excel_file.xlsx">Dataset</a>

# File:
- pizza_sales.csv
The dataset contains one year of pizza sales transactions, including order date, pizza category, pizza size, quantity sold, unit price, and total price.

 ## Questions (KPIs)
 
- What is the total revenue generated from pizza sales?
- What is the average order value?
- How many pizzas were sold during the selected period?
- How many total customer orders were placed?
- What is the average number of pizzas sold per order?
- Which day of the week receives the highest number of orders?
- Which month records the highest and lowest number of orders?
- Which pizza category contributes the highest percentage of total sales?
- Which pizza size generates the highest revenue?
- Which pizza category sells the highest number of pizzas?
- Which are the Top 5 pizzas by revenue?
- Which are the Bottom 5 pizzas by revenue?
- Which are the Top 5 pizzas by quantity sold?
- Which are the Bottom 5 pizzas by quantity sold?
- Which are the Top 5 pizzas by total orders?
- Which are the Bottom 5 pizzas by total orders?

## Process

- Imported the pizza sales dataset into Microsoft Excel and SQL Server.
- Cleaned and prepared the data by removing duplicates and validating records.
- Used SQL queries to calculate business KPIs and sales metrics.
- Created Pivot Tables to summarize sales performance.
- Designed Pivot Charts to visualize trends and category-wise performance.
- Added Slicers and Timeline filters for interactive dashboard analysis.
- Combined all visualizations into a single interactive dashboard.
- Verified dashboard accuracy by comparing SQL query results with Excel reports.
